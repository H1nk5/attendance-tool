# 考勤表整理工具

从总考勤表中按姓名列表筛选人员，生成独立考勤表。基于 WPS COM 自动化，保留原始格式和列宽。

## 功能

- 从总考勤表读取全部姓名
- 按名单筛选指定人员
- 复制时保留格式、列宽、行高
- 未匹配人员会提示
- GUI 界面，操作简单

## 依赖

```
pip install pywin32
```

需要安装 WPS Office。

## 使用

1. 运行 `python 考勤表整理工具.py`
2. 选择总考勤表文件
3. 输入或读取人员名单
4. 点击"生成考勤表"
